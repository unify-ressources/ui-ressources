## What's UnifyUI

UnifyUI is a collection of blocks and components designed to work seamlessly with UnoCSS. 
<div>
Designed to be attractive, functional, lightweight, flexible, and easy to use. 
</div>
<div>
Unify-UI's goal is to provide developers with the tools they need to create beautiful and engaging user interfaces with ease. 
</div>

## What is UnoCSS?

UnoCSS is the instant atomic CSS engine, that is designed to be flexible and extensible.

The core is un-opinionated, and all the CSS utilities are provided via presets.


<Callout type="note">
  UnifyUI is free and Open-source, every contribution is welcome... UnifyUI is currently in beta version.
</Callout>
