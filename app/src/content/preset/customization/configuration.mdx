PresetUI allows you to customize the appearance of your components as needed, including changing default shades and sizes. As mentioned in the [Introduction](/preset/introduction), PresetUI features Component-Level Theming.
In most cases, you won't need to update the configuration, as the default settings are suitable for many scenarios.



## Configuration

Some components share the same configuration. You can decide if you want your components to have their own configuration or a shared one.

<Callout type="note">
  **Info :**
  The idea to add a configuration part is mainly inspired by the way NuxtUI makes it easier to customize each UI element. We've oriented this towards controlling the output CSS. Every change in the configuration will be reflected in the final CSS output.
</Callout>

<BlockCode>
```js
// uno.config.ts
import { presetUI } from "@unifydev/preset-ui"

export default defineConfig({
  // ...config
  presets: [
    //other presets
    presetUI({
      
	    components:{
        // configure each components styles
      },
      // shared configuration
      ui:{
        globalElement:{
            //default configuration for solib UI shades
            solid:{
                bgShade:"400",
                textShade:"900",
                // add dark if needed
                dark:{
                    bgShade:"600",
                    textShade:"100",
                }
            }
        },

      }

    })
  ],
});
```
</BlockCode>

<Callout type="note">
  UnifyUI Preset is generated on demand. This configuration will work for all colors. 
  We are planning to make it possible to configure each color independently in future updates.
</Callout>

## External Configuration

You can separate things by creating a separate `ui.config.ts` file for your configuration

<CodeGroupBox>
<BlockCode>
```js
// ui.config.ts
import { type presetUIOptions } from "@unifydev/preset-ui"

export const myConfig:presetUIOptions = {
    // your configuration here
}

```
</BlockCode>
<BlockCode>
```js
// uno.config.ts
import { presetUI } from "@unifydev/preset-ui"
import { myConfig } from "./ui.config.ts"

export default defineConfig({
  // ...config
  presets: [
    presetUI(myConfig)
  ],
});
```
</BlockCode>
</CodeGroupBox>
