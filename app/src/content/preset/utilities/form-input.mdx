## Form Input

To use it just add `form-input` utility class to your input, this class contains base styles.

### Styles

1. Background

The Input component comes with the following styles for background: `form-input-gray`, `form-input-high-gray`, `form-input-higher-gray`, `form-input-light-gray`

2. Border

The Input component comes with the following styles for border: `bordered`, `bordered-light`, `bordered-high`, `bordered-higher`

### Sizes

The Input component supports the following sizes : `2xs`, `xs`, `sm`, `md`, `lg`, `xl`

### Examples

<BlockCode>
```html
<input type="text" class="form-input form-input-md form-input-bordered form-input-gray form-input-placeholder-light">
```
</BlockCode>

<Callout type="note">
  Using form input utilities require having [@julr/unocss-preset-forms]() Preset installed.
</Callout>
