## What is PresetUI

PresetUI is a UI component library designed to streamline your development process. Built on top of UnoCSS, it provides a collection of pre-built, customizable components that you can easily integrate into your projects.

Preset UI offers a comprehensive collection of utility classes, making it easy to apply styles directly to your elements. 
This library is generated on demand and follows atomic CSS principles, ensuring efficient and optimized CSS.


###  Utility Classes

Preset UI comes with utility classes. These classes are applied directly to elements, allowing for granular control over your design. The library follows an Atomic and Generate On Demand approach, meaning it generates CSS only for the utilities you use. This results in a more efficient and optimized output, making your projects lighter and faster.

### Benefits

- **On-Demand Generation**: The CSS is generated only for the utilities you use, ensuring efficiency and reducing bloat.
- **Atomic Design**: Following atomic design principles, Preset UI provides small, reusable utility classes that can be combined to create complex styles.
- **Customizable**: Easily customize your components with a variety of themes and settings to match your unique style and brand.
- **Light/Dark Mode Integration**: Seamlessly switch between light and dark modes, ensuring your UI looks great in any environment. By using Preset UI and its utility classes, you can streamline your development process, reduce CSS bloat, and create stunning, customizable UIs with ease.
- **Freedom on Appearance :** Customize your styles to match your preferences. Whether you prefer only dark style, light and dark styles, or only light styles, Unify Preset accommodates your needs. The generated CSS will contain only what you decide to have.
- **Efficient Scaling :** Our atomic library ensures your project remains lightweight and fast, no matter its size.
- **Consistent Yet Flexible :** Achieve a consistent look and feel without sacrificing the creativity and uniqueness of individual components.
  
### Variants
Preset UI comes with various variants for each component, allowing for different styles and appearances.

- **Solid Variants :**
To use this variant, simply add the utility class `ui-solid-colorName` to an item.

- **Soft Variants :**
To use this variant, simply add the utility class `ui-soft-colorName` to an item.

- **Subtle Variants :**
To use this variant, simply add the utility class `ui-subtle-colorName` to an item.

- **Outline Variants :**
To use this variant, simply add the utility class `ui-outline-colorName` to an item.
