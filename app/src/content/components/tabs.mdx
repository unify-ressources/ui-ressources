## Initialisation

<Callout type="note">
Note that this component requires JS, by default we're using `@flexilla/tabs` component, else you can skip this step if you've already `@flexilla/flexilla`.
</Callout>

<Steps>
<Step>
### Installation
<BlockCode shellStyle>
```bash
npm i @flexilla/tabs
```
</BlockCode>
</Step>
<Step>
### Usage
You can globaly init all Tabs or init separately every tabs Element
<BlockCode>
```js
import { Tabs } from "@flexilla/tabs"
// init Tabs for all Element with data-fx-tabs attribute
Tabs.autoInit() 
```
</BlockCode>
</Step>
</Steps>
