## Usage

<Callout type="note">
Note that this component requires JS, by default we're using `@flexilla/tooltip` component, else you can skip this step if you've already `@flexilla/flexilla`.
</Callout>

<Steps>
<Step>
### Installation
<BlockCode shellStyle>
```bash
npm i @flexilla/tooltip
```
</BlockCode>
</Step>
<Step>
### Usage
You can globaly init all Popover/Tooltip or init separately every modal Element
<BlockCode>
```js
import { Tooltip } from "@flexilla/tooltip"
// init modal for all Element with data-popover attribute
Tooltip.autoInit('[data-popover]') 
```
</BlockCode>
</Step>
</Steps>

## Trigger on Hover

<PreviewBox name="popover/OnHover">
```html

    <button data-fx-tooltip-trigger aria-controls="popper" class="btn btn-md btn-white ease-linear rd-lg relative">
        Open On Hover
    </button>
    <div aria-labelledby="popper" data-fx-popper class="u-fx-popper w-60 p-4 bg-high rd-md invisible op0 fx-open:visible fx-open:op100">
        <div class="flex h-40 rounded bg-high">

        </div>
    </div>
</div>
```
